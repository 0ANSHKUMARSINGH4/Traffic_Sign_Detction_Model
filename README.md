
# 🇮🇳 Universal Indian Traffic AI (Zero-Shot)

A "Company Grade" Zero-Shot detection system powered by **YOLO-World**. 
This repository was completely upgraded from traditional CNN models to a state-of-the-art Open-Vocabulary architecture.

## 🚀 Capabilities
- **Indian Road Context:** Detects *Cows*, *Auto Rickshaws*, *Potholes*, *Helmets*, *Traffic Signs*, and *Speed Bumps*.
- **Zero-Shot Learning:** No manual labeling required. Detects objects based on text definitions.
- **Robustness:** Handles chaotic scenes using Test Time Augmentation (TTA).

## 🛠️ Usage
```bash
pip install ultralytics opencv-python
python detect_indian_traffic.py
